A cbz reader for php

Requirements

  • PHP >= 5.3
  • Composer

Installation

  • Clone the repository: git clone https://github.com/coverslide/php-cbz-reader
  • Get Composer (http://getcomposer.org)
  • Install dependencies composer install
  • Copy config.ini.example to config.ini cp config/config.ini{.example,}
  • Modify cbz.dir entry in config/config.ini
  • Point your document root to the web/ directory
